# WebStudio

Responsive landing page developed using HTML5 and CSS3.

## Features

- Semantic HTML markup
- Responsive layout
- Flexbox-based positioning
- CSS custom properties (variables)
- Optimized images
- Accessible navigation and content structure
- SVG icon sprite with CSS-controlled fill
- CSS transitions on interactive elements (buttons, social links)
- Hero section with gradient overlay on background image
- Footer social media links with circular icon buttons

## Technologies

- HTML5
- CSS3
- Google Fonts (Roboto, Raleway)
- Prettier
